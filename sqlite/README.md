# SQLite Folder

This folder contains SQL scripts used to analyze the dataset.

## Contents

- Basic `SELECT` statements
- Data filtering using `WHERE`
- Sorting using `ORDER BY`
- Grouping with `GROUP BY`
- Aggregate functions such as `COUNT`, `SUM`, `AVG`, `MIN`, and `MAX`

The SQL queries demonstrate how structured query language can be used to extract meaningful insights from the dataset.
