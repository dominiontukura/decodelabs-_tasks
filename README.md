# Data Analytics Internship Project

## Description

This repository contains solutions for a Data Analytics internship assignment covering the complete analytics workflow:

- Data Cleaning & Preparation
- Exploratory Data Analysis (EDA)
- SQL Data Analysis
- Data Visualization

The project demonstrates practical skills in cleaning raw datasets, identifying trends, generating insights, writing SQL queries, and communicating findings through visualizations.

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/data-analytics-internship-project.git
```

### 2. Navigate into the project

```bash
cd data-analytics-internship-project
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open the notebooks in order:

1. `01_data_cleaning.ipynb`
2. `02_exploratory_data_analysis.ipynb`
3. `03_sql_analysis.ipynb`
4. `04_visualization.ipynb`
